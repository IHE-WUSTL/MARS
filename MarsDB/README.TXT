The MarsDB project is a library project, generating the MarsDB.jar file used by
the IIGFeed, Mars, and MarsReports projects. It contains the following
packages:

edu.wustl.mir.mars.db - The Hibernate Entity classes for the mars database.

edu.wustl.mir.mars.hibernate - Utility classes used in Hibernate processing and
log4j log processing.

edu.wustl.mir.mars.util - General utility classes.

edu.wustl.mir.mars.syngo - The pseudo Entity class Patient.java, representing
the results of a query against the Syngo RIS database, using the Schedule
project.
